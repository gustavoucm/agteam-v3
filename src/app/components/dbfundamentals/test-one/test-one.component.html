Evaluación Unidad 1

Este es un test para evaluar tus conocimientos en la Unidad 1. Contesta según consideres correcto

1.- Unidad minima de información y que carece de sentido por si sola

2.- Conjunto de datos pertenecientes a un mismo contexto y almacenados sistemáticamente para su posterior uso

3.- Característica o rasgo de un tipo de entidad que describe la entidad

4.- Objeto o concepto del mundo real acerca del cual se recoge información, y que puede diferenciarse de otros objetos, incluso si son de su misma clase.

5.- Tipo de modelo de base de datos que se asemeja a la estructura de un árbol

6.- Modelo de base de datos en que las entidades se representan como nodos y sus relaciones son las líneas que los unen.

7.- Modelo de base de datos en la que se almacena un registro con un enlace a otros registros.

8.- Ejemplo de Sistema Gestor de Bases de Datos NoSQL.

9.- Nivel de la arquitectura de un SGBD que esta más cercano al usuario y que describe la parte de la BD que interesa a un grupo de usuarios .

10.- Nivel de la arquitectura de un SGBD que describe las entidades, atributos, relaciones, operaciones de los usuarios y restricciones, ocultando los detalles de las estructuras físicas de almacenamiento
